# SistemaInventarioYPuntoVenta

Jesus Eduardo Cruz Hernandez

Proyecto parcial 2

Este proyecto implementa un sistema de gestión de inventario en C++ que cumple con todas las funcionalidades solicitadas, aplicando los patrones de diseño Factory Method y Observer, además del uso de archivos JSON para guardar los datos.

Descripción
El sistema permite cargar productos desde un archivo JSON, realizar ventas y compras, agregar nuevos productos de forma dinámica y enviar alertas automáticas cuando el stock es bajo.

Patrones Utilizados:
Factory Method
Observer

Funcionalidades Cumplidas:

Carga de productos desde JSON
Mostrar inventario completo
Venta y compra de productos
Agregar nuevos productos con Factory
Guardar inventario actualizado
Alertas automáticas por bajo stock (Observer)
Notificaciones de operaciones realizadas
